# Testimonials Grid Section

A responsive and interactive testimonials section inspired by Frontend Mentor's design.

## Features

- **Responsive Grid Layout:** Works perfectly on desktop, tablet, and mobile.
- **Hover Effect:** Cards slightly enlarge on hover for a dynamic feel.
- **Modal Popup:** Click on any card to view full testimonial in a popup.
- **Dark Mode Toggle:** Switch between light and dark themes.
- **Clean UI:** Modern design with smooth transitions and readable typography.

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Transitions)
- JavaScript (Event handling, DOM manipulation)

## Installation

1. Clone the repository:
  https://afayad323-lgtm.github.io/testimonials/

   
Open index.html in your browser.

Usage
Hover over any testimonial card for a subtle animation.

Click on a card to open the full testimonial in a modal.

Use the Dark Mode button to switch themes.
